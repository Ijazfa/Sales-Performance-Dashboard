This repository contains the Power BI file for the Sales Performance Dashboard, a project designed to provide a detailed analysis of sales metrics. The dashboard offers various visualizations to track and understand sales performance across different dimensions such as region, product, and time.

✨ Features:

📊 Interactive visualizations for in-depth analysis
🔄 Real-time data updates
🔍 Filters and slicers to customize views
🧠 Dashboard optimized for business decision-making
This project highlights my proficiency in data visualization and my ability to create meaningful dashboards that drive business insights.
